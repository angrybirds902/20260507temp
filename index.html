Q1(a)

Sampling Methods 抽樣方法

🔍 咩係 Sampling？

你唔可能問全世界每一個人嘅意見，所以你揀一部分人嚟代表成個群體，呢個過程就叫 Sampling（抽樣）。

1️⃣ Simple Random Sampling（簡單隨機抽樣）

English: Every member of the population has an equal chance of being selected.
  • 白話：好似抽獎咁，每個人被抽中嘅機會一樣
  • 例子：從所有註冊醫生入面隨機揀 50 個
  • 口訣：「人人平等，隨機抽」

2️⃣ Stratified Sampling（分層抽樣）

English: Population is divided into subgroups (strata), then random samples are drawn from each stratum.
  • 白話：先分組（例如按部門），再喺每組入面各自隨機抽人
  • 例子：銀行有 Marketing、Research、Finance 三個部門，分別隨機抽 30、20、50 人
  • 口訣：「先分層，再抽樣」

3️⃣ Systematic Sampling（系統抽樣）

English: Select every k-th member from a list or sequence.
  • 白話：每隔固定人數抽一個，好似排隊每 8 個人揀 1 個咁
  • 例子：每第 8 個入學校嘅學生就問佢最鍾意邊個老師
  • 口訣：「每隔 K 個抽一個」

✅ 答案速查

情境答案
(i) 隨機揀 50 個醫生Simple Random Sampling
(ii) 按部門各自抽人Stratified Sampling
(iii) 每第 8 個學生Systematic Sampling

🧠 考試 Tips

  • 見到「randomly selected from the whole group」→ Simple Random
  • 見到「divided into groups / departments, then randomly select from each」→ Stratified
  • 見到「every k-th」→ Systematic

Normal Distribution 常態分佈

題目：ABC 航空 HK→台灣飛行時間 X ~ N(90, 15²)，即 μ = 90 min，σ = 15 min

🔔 基礎概念

  • 鐘形曲線（Bell Curve），左右對稱
  • 大部分數據集中喺 mean 附近，愈遠愈少
  • 由 μ (mean) 同 σ (standard deviation) 決定

🔑 Z-Score 公式

Z-Score
Z = (X − μ) / σ
  • Z 嘅意思：呢個數值距離 mean 有幾多個 standard deviation
  • Z ~ N(0, 1)，即 Standard Normal Distribution
  • 成條曲線總面積 = 1；左半邊 = 右半邊 = 0.5

📌 Q2(a)：P(X < 81) = ?

  • 1計 Z-score：Z = (81 − 90) / 15 = −9/15 = −0.6
  • 2查表：P(0 < Z < 0.6) = 0.2257
  • 3P(Z < −0.6) = 0.5 − 0.2257 = 0.2743
💡Z 係負數 → 喺 mean 左邊 → 面積 = 0.5 − 表中值

📌 Q2(b)：搵 M，使得 P(X > M) = 0.20

  • 1P(X > M) = 0.20 → P(X < M) = 0.80
  • 2P(0 < Z < z) = 0.80 − 0.50 = 0.30 → 反查表 → Z ≈ 0.84
  • 3M = μ + Z × σ = 90 + 0.84 × 15 = 102.6 minutes

📌 Q2(c)：Binomial Distribution（結合 Normal）

題目：隨機抽 20 班機，求剛好 5 班飛行時間少過 81 分鐘嘅概率
  • 由 part (a)：p = P(X < 81) = 0.2743
  • Y ~ Bin(n = 20, p = 0.2743)
P(Y = 5) = C(20,5) × (0.2743)⁵ × (0.7257)¹⁵ = 15504 × 0.001553 × 0.01026 ≈ 0.2470
Binomial 條件： ① 固定次數 n ② 每次只有兩個結果 ③ 每次概率 p 相同 ④ 各次獨立

📌 Q2(d)：Linear Transformation（線性變換）

升級後飛行時間減少 8%，即新時間 = 0.92X，求新 E 同 SD
Y = aX + b 嘅規則
E(aX + b) = a·E(X) + b SD(aX + b) = |a|·SD(X) Var(aX + b) = a²·Var(X)
E(0.92X) = 0.92 × 90 = 82.8 minutes SD(0.92X) = 0.92 × 15 = 13.8 minutes

🧠 超重要記憶表

變換 Y = aX + bE(Y)SD(Y)Var(Y)
公式aE(X) + b|a|·SD(X)a²·Var(X)
加減常數 b影響 mean唔影響 SD唔影響 Var
乘除常數 a影響 mean影響 SD影響 Var

📝 Q2 總結

Part考點核心公式
(a)Normal → Z-score → 查表Z = (X − μ) / σ
(b)反查 Normal TableX = μ + Zσ
(c)Normal + Binomial 結合用 (a) 嘅概率做 Binomial 嘅 p
(d)Linear TransformationE(aX) = aE(X), SD(aX) = |a|SD(X)

Price Index 價格指數

題目：水果 (Apple, Orange, Pear, Strawberry) 2019–2021 年嘅 Price & Quantity 數據

🔍 咩係 Price Index？

  • 用嚟比較唔同年份嘅價格水平
  • 以某一年做 Base Year（基期），設為 100
  • Index > 100 → 價格上升（通脹 Inflation）
  • Index < 100 → 價格下降（通縮 Deflation）

📌 兩種 Price Index 公式

Laspeyres Price Index（L = Last/舊數量）
PI_L = [Σ(q₀ × pₜ) / Σ(q₀ × p₀)] × 100 — 用 Base Year 嘅 Quantity (q₀)
Paasche Price Index（P = Present/新數量）
PI_P = [Σ(qₜ × pₜ) / Σ(qₜ × p₀)] × 100 — 用 Current Year 嘅 Quantity (qₜ)
LaspeyresPaasche
Quantity 用邊年？Base Year (q₀)Current Year (qₜ)
分子Σ q₀ × pₜΣ qₜ × pₜ
分母Σ q₀ × p₀Σ qₜ × p₀
記法L = 舊 (Last)P = 新 (Present)
💡兩個公式共同點：分子永遠用 current price (pₜ)，分母永遠用 base price (p₀)，最後都要 × 100

🧠 Interpretation 模板（考試直接用）

"Based on the [Laspeyres / Paasche] Price Index, the overall price level in [current year] has [increased / decreased] by approximately [Index − 100]% compared to the base year [base year], using [base year / current year] quantities as weights."

  1. 睇清楚邊年做 Base Year！唔一定係最早嗰年
  2. Laspeyres 同 Paasche 唔好撈亂 — 記住 L=舊數量、P=新數量
  3. 最後一定要 × 100
  4. Interpretation 一定要寫，唔好淨係寫個數字

Descriptive Statistics 描述統計量

數據（排好序）： 93, 96, 99, 102, 105, 106, 108, 115, 120, 121, 131, 146  (n = 12)

📌 Q4(a)：Sample Mean 同 Sample SD

Sample Mean
x̄ = Σx / n = 1342 / 12 = 111.83 minutes
Sample Standard Deviation（分母用 n−1！）
s = √[(Σx² − nx̄²) / (n−1)] Σx² = 152,738 s = √[(152738 − 12 × 111.83²) / 11] = √242.42 = 15.57 minutes
⚠️Sample SD 分母係 n−1，Population SD 分母先係 n！

📌 Q4(b)：Range 同 IQR

Range = Max − Min = 146 − 93 = 53 minutes
Quartile 位置公式
Q₁ position = (1/4)(n+1) = (1/4)(13) = 3.25 Q₁ = x₃ + 0.25 × (x₄ − x₃) = 99 + 0.25 × 3 = 99.75 Q₃ position = (3/4)(n+1) = (3/4)(13) = 9.75 Q₃ = x₉ + 0.75 × (x₁₀ − x₉) = 120 + 0.75 × 1 = 120.75 IQR = Q₃ − Q₁ = 120.75 − 99.75 = 21 minutes

📌 Q4(c)：Skewness 偏態

Q₂ (Median) = (x₆ + x₇) / 2 = (106 + 108) / 2 = 107 Q₃ − Q₂ = 120.75 − 107 = 13.75 Q₂ − Q₁ = 107 − 99.75 = 7.25 Since Q₃ − Q₂ (13.75) > Q₂ − Q₁ (7.25) → Positively Skewed
情況結論
Q₃ − Q₂ > Q₂ − Q₁Positive skew ➡️ 右邊較散
Q₃ − Q₂ < Q₂ − Q₁Negative skew ⬅️ 左邊較散
Q₃ − Q₂ = Q₂ − Q₁Symmetric ↔️ 對稱
💡口訣：「右大正偏，左大負偏」

Q5

Confidence Interval 置信區間

🔍 基礎概念

我哋唔可能知道成個 population 嘅真正 mean 或 proportion，所以我哋用 sample data 去估計一個範圍：

Point Estimate ± Margin of Error = 估計值 ± (Critical Value × Standard Error)

🧠 用 Z 定 t？

情況用邊個？
σ 已知 (known)Z
σ 未知 (unknown)，用 s 代替t（df = n−1）
估 proportionZ（永遠）

🧠 必背 Z 值

Confidence Levelαα/2Z value
90%0.100.051.645
95%0.050.0251.96
99%0.010.0052.576

📌 Q5(a)：CI for Mean (σ unknown → t)

Blood test time，σ unknown，n = 28，x̄ = 130 min，s = 25 min，90% CI
  • 1df = n − 1 = 27，α/2 = 0.05 → t₀.₀₅,₂₇ = 1.703
  • 2SE = s / √n = 25 / √28 = 4.7245
  • 3ME = t × SE = 1.703 × 4.7245 = 8.04
  • 4CI = 130 ± 8.04 = (121.96, 138.04) minutes
"We are 90% confident that the population mean time required to finish the blood test is between 121.96 and 138.04 minutes."

📌 Q5(b)：CI for Proportion (永遠用 Z)

n = 200，defective = 25，p̂ = 25/200 = 0.125，95% CI
  • 1Z₀.₀₂₅ = 1.96
  • 2SE = √[p̂(1−p̂)/n] = √[0.125 × 0.875 / 200] = 0.02339
  • 3ME = 1.96 × 0.02339 = 0.04585
  • 4CI = 0.125 ± 0.04585 = (0.0792, 0.1709) 即 (7.92%, 17.09%)
"We are 95% confident that the population proportion of defective items is between 7.92% and 17.09%."

🧠 考試做題流程

  • 1判斷用 Z 定 t（σ known → Z；σ unknown → t；proportion → Z）
  • 2搵 Critical Value（查表或背）
  • 3計 Standard Error (SE)
  • 4計 Margin of Error (ME) = Critical Value × SE
  • 5CI = Point Estimate ± ME
  • 6寫 Interpretation！
⚠️
常見陷阱：
  • σ unknown 一定用 t，唔好用 Z！
  • t-distribution 要寫 degrees of freedom = n − 1
  • Proportion 永遠用 Z
  • Interpretation 一定要寫！

Discrete Probability Distribution 離散概率分佈

題目：小城市每日交通意外數量 X 的概率分佈
x012345
P(X=x)0.10.20.450.150.05k

📌 最重要規則

ΣP(X = x) = 1  （所有概率加埋一定等於 1）

Q6(a)：搵 k

0.1 + 0.2 + 0.45 + 0.15 + 0.05 + k = 1 0.95 + k = 1 k = 0.05

Q6(b)：E(X)（期望值）

E(X) = Σ x·P(X=x)
= 0×0.1 + 1×0.2 + 2×0.45 + 3×0.15 + 4×0.05 + 5×0.05 = 0 + 0.20 + 0.90 + 0.45 + 0.20 + 0.25 = 2.0

Q6(c)：Var(X) 同 σ(X)

Var(X) = E(X²) − [E(X)]²
E(X²) = 0×0.1 + 1×0.2 + 4×0.45 + 9×0.15 + 16×0.05 + 25×0.05 = 0 + 0.20 + 1.80 + 1.35 + 0.80 + 1.25 = 5.40 Var(X) = 5.40 − (2.0)² = 5.40 − 4.00 = 1.40 σ(X) = √1.40 = 1.1832
💡口訣：「平方嘅期望 減 期望嘅平方」

Q6(d)：30 日內預期幾多日多過 3 宗意外

P(X > 3) = P(X=4) + P(X=5) = 0.05 + 0.05 = 0.10 Expected days = 30 × P(X > 3) = 30 × 0.10 = 3 days

🧠 做題標準流程

  • 1確認 ΣP = 1，搵 missing value
  • 2整一個表，列出 x, P(X=x), x·P, x²·P
  • 3加晒 x·P → 得到 E(X)
  • 4加晒 x²·P → 得到 E(X²)
  • 5Var(X) = E(X²) − [E(X)]²
  • 6σ(X) = √Var(X)
⚠️
「more than 3」唔包括 3！
P(X > 3) = P(X=4) + P(X=5) ≠ P(X ≥ 3)
